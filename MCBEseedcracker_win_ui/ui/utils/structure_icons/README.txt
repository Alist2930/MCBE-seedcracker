| Woodland Mansion | 林地府邸 | `woodland-mansion.png` | https://zh.minecraft.wiki/images/EnvSprite_woodland-mansion.png |
| End City | 末地城 | `end-city.png` | https://zh.minecraft.wiki/images/EnvSprite_end-city.png |
| Ocean Monument | 海底神殿 | `ocean-monument.png` | https://zh.minecraft.wiki/images/EnvSprite_ocean-monument.png |

| Nether Fortress | 下界要塞 | `nether-fortress.png` | https://zh.minecraft.wiki/images/EnvSprite_nether-fortress.png |

## 下载方法

1. 在浏览器中打开上方的 URL
2. 右键图片 → 另存为
3. 保存到 `ui/utils/structure_icons/` 目录
4. 确保文件名与上表一致

## 注意

- 如果没有下载图标文件，程序会自动使用纯色方块作为替代
- 图标尺寸建议 16x16 像素
- 支持 PNG 格式
